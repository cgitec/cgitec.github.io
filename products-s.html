S-Series

High-Efficiency Spectrometer

S100

S시리즈는 높은 효율과 정확성을 자랑하는 산업용 스펙트로미터입니다. 다양한 산업 현장에서 신뢰성 있는 방사선 분석 데이터를 제공합니다.

Key Features

  • 고효율 중성자 및 감마선 검출
  • 견고한 산업용 디자인
  • 네트워크 연결을 통한 원격 모니터링
  • 사용자 친화적인 인터페이스

Applications

  • 산업체 공정 제어 및 모니터링
  • 환경 감시 네트워크
  • 폐기물 분류 및 처리 시설

Features

  • Practical high-performance gamma-ray spectrometer
  • View trends in real time over network
  • Compact and light weight
  • Industry-leading CZT efficiency

Applications

  • Isotopic characterization of pipes and valves
  • Isotopic trend analysis
  • Outage monitoring
  • Waste quantification
